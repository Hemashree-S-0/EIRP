# 🌿 Environmental Incident Reporting Portal (EIRP)

A full-stack web application that enables citizens to report environmental incidents such as air pollution, water pollution, waste dumping, deforestation, and chemical leaks. The system allows officers and administrators to track, manage, and resolve incidents through an intuitive dashboard.

## 🚀 Tech Stack

- **Frontend:** HTML, CSS, JavaScript, Bootstrap
- **Backend:** Node.js, Express.js
- **Database:** MySQL
- **Authentication:** JWT, bcrypt
- **Others:** Multer, Chart.js, Leaflet.js

## ✨ Features

- User Registration & Login
- Report Environmental Incidents
- Upload Images
- Track Incident Status
- Citizen, Officer & Admin Dashboards
- Analytics & Reports
- Search, Filter & Pagination
- Responsive UI
